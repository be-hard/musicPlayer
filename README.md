# musicplayer
```
基于Vue，Vuex，Vue Router的单页面应用--音乐播放器
```
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
